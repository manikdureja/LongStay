# LongStay Local App

This repository now runs fully locally without Base44. The app uses a local mock client for authentication and data storage.

## Prerequisites

1. Clone the repository using the project's Git URL.
2. Navigate to the project directory.
3. Install dependencies:

```bash
npm install
```

## Run Locally

Start the frontend with Vite:

```bash
npm run dev
```

Open the local URL printed by Vite in your browser.

## Local Credentials

Use one of the seeded accounts to sign in:

- admin@longstay.local / admin123
- host@longstay.local / host123
- guest@longstay.local / guest123

You can also register a new local account using the sign-up flow.
